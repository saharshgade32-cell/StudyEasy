# 📚 StudyEasy by Saharsh Gade

A complete school companion tool that helps students manage their studies with privacy and ease.

## ✨ Features

- **🎯 Flashcards** - Create and study flashcards
- **✓ Homework Tracker** - Manage assignments with due dates
- **📊 Grade Calculator** - Track your scores and GPA
- **⏱️ Pomodoro Timer** - Study timer with presets
- **🔢 Math Solver** - Quick calculations and formulas
- **📝 Notes** - Capture class notes organized by subject
- **📅 Schedule** - Manage your class schedule
- **🎓 GPA Calculator** - Calculate cumulative GPA automatically
- **📚 Dictionary** - Build your vocabulary and search definitions

## 🔐 Privacy

- ✅ **Account System** - Create an account to keep your data private
- ✅ **All data is stored locally** - Nothing goes to servers
- ✅ **Each user is separate** - Multiple users can use the same device
- ✅ **Your data, your control** - Only you can see your information

## 🚀 How to Use

1. Open the website
2. Create an account (username + password)
3. Start using all the tools
4. Your data is automatically saved

## 📱 Works on

- Desktop computers
- Tablets
- Mobile phones
- Any device with a web browser

## 🛠️ Technical Details

- Single HTML file (no installation needed)
- Uses browser's local storage for data
- No external servers or accounts required
- Works offline after first load
